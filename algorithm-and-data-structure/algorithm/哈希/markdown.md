## 特性

### 散列函数特性：

- 冲突少
- 计算难度低(节约计算成本)
- 充分分散数据项(节约内存)

### 数据摘要函数特性：

- 压缩性(散列值长度固定)
- 易计算性(计算散列值容易，逆推不可能)
- 抗修改性(原数据小变动引起散列值大变动)
- 抗冲突性(已知原数据和散列值，难以伪造相同散列值的其他原数据)



## 散列函数设计

### 折叠法：

- 将数据项按照位数分为若干段，
  再将几段数字相加，
  最后对散列表大小求余，得到散列值

- 使用隔数翻转(如76->67)来进行微调

### 平方取中法：

先平方，再取中间两位，再对长度求余

### 字符类：

- 取各字符ascii码累加
- 乘权重因子区分变位词
